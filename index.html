Automated Footfall Mapping

My summer research internship, performed for the Victoria and Albert Waterfront in Cape Town, South Africa. I used computer vision techniques on security camera footage from the Waterfront to create a system to count the number of pedestrians passing through the area. This project was my first real experience of applying machine learning to a real-world problem.
